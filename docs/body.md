## Controls (By default)
- Gamepad
  - Directional pad: Move character/between options
  - B/O button: Select
  - A/X button: Cancel
  - Start/Options button: Menu

- Keyboard & Mouse
  - Arrow keys: Move character/between options
  - Z: Select
  - X: Cancel
  - C/Esc: Menu
  - Mouse click: Select button/target (combat)

- Debug keys:
  - Esc: Exit game (on title/Dev room)
  - F1: Show colliders
  - F2: Lose the game
  - F3: Trigger combat
  - F4: Save request
  - F5: Load request
  - F7: Complete Quest
  - F10: God mode
    - No collisions
    - No event triggers
    - No encounters
    - No damage in combat
    - Shift: Run
